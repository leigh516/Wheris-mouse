# Wheris Mouse (커서 찾기)

마우스 휠(중간) 버튼을 클릭하면 커서 위치를 2초간 시각적으로 강조해주는 Windows 유틸리티입니다.

## 효과

- **점멸 링**: 커서 주변에 빨강·주황·노랑 3겹 링이 커졌다 작아지기를 반복
- **커서 확대**: 실제 커서 위에 3배 크기 화살표 오버레이가 실시간으로 따라다님

## 요구사항

- Windows 10 / 11
- Python 3.x (표준 라이브러리만 사용: `tkinter`, `ctypes`, `threading`)

## 실행 방법

### 직접 실행

```bash
python mouse_finder.py
```

### 백그라운드 실행 (콘솔 창 없음)

`실행.bat`를 더블클릭하면 `pythonw`로 콘솔 없이 백그라운드에서 실행됩니다.

## 사용법

| 동작 | 설명 |
|------|------|
| 휠(중간) 버튼 클릭 | 커서 강조 효과 2초간 재생 |
| 화면 우측 상단 **X** 클릭 | 프로그램 종료 |

## 다중 모니터 지원

가상 화면 전체 좌표계를 사용하므로 멀티 모니터 환경에서도 정상 동작합니다.
고DPI(예: 150%) 환경에서도 커서 오버레이 위치가 정확하게 표시됩니다.
